# Анализ ценообразования подержанных автомобилей

Эконометрический проект, посвящённый исследованию факторов, определяющих стоимость подержанных автомобилей на российском рынке.

В работе выполнены очистка и преобразование данных, разведочный анализ, построение интерпретируемой OLS-модели, диагностика гетероскедастичности и проверка возможной эндогенности пробега. Для получения более устойчивых оценок дополнительно применён метод инструментальных переменных IV/2SLS.

## Цель проекта

Определить, какие характеристики автомобиля статистически связаны с его ценой, оценить направление и величину этих связей и проверить устойчивость выводов к нарушениям предпосылок классической линейной регрессии.

## Данные

Исходный набор содержит **2 094 объявления** о продаже подержанных автомобилей.
После очистки и формирования итоговой спецификации в основной регрессии использовалось **2 026 наблюдений**.

Признаки включают:

- марку и модель автомобиля;
- год выпуска;
- мощность двигателя;
- объём двигателя;
- цену;
- город размещения объявления;
- пробег;
- тип топлива;
- тип коробки передач;
- цвет;
- расположение руля;
- количество владельцев.

Целевая переменная — цена автомобиля. Для эконометрического анализа использовался логарифм цены.

## Этапы работы

### 1. Подготовка данных

В рамках предобработки:

- приведены числовые признаки к единому формату;
- обработаны пропуски;
- удалены некорректные и экстремальные наблюдения;
- преобразован пробег;
- сформирован возраст автомобиля;
- объединены редкие категории цветов;
- города сгруппированы в укрупнённые региональные категории;
- созданы dummy-переменные для категориальных признаков;
- сформирован объединённый признак марки и модели;
- для цены и пробега использованы логарифмические преобразования.

### 2. Базовая OLS-модель

Построена множественная линейная регрессия для логарифма цены.
Итоговая модель объясняет около **84% вариации логарифма цены**.

### 4. Диагностика модели

После оценки OLS были проверены ключевые предпосылки:

- статистическая значимость коэффициентов;
- мультиколлинеарность;
- гетероскедастичность;
- корректность функциональной формы;
- устойчивость стандартных ошибок.

Так как была выявлена гетероскедастичность, для интерпретации использовались робастные стандартные ошибки HC1.

### 5. Проверка эндогенности пробега

Пробег может быть эндогенным из-за:

- ненаблюдаемого технического состояния;
- интенсивности эксплуатации;
- качества обслуживания;
- ошибок измерения;
- одновременной связи пробега и цены с пропущенными характеристиками автомобиля.

Для проверки применён тест Ву—Хаусмана.
Результат теста:
- статистика: **15.918**;
- p-value: **0.0001**.
- 
Нулевая гипотеза об экзогенности пробега отвергается, поэтому дополнительно была оценена IV/2SLS-модель.

### 6. IV/2SLS-модель

В качестве эндогенной переменной использовался логарифм пробега.
В качестве инструмента рассматривался транспортный индекс региона. Также была протестирована спецификация с дополнительным инструментом — ценой топлива.

Для модели с транспортным индексом:

- partial F-statistic первого этапа: **12.518**;
- p-value: **0.0004**;
Инструмент статистически значим на первом этапе, однако его объясняющая способность невысока. Поэтому IV-результаты следует интерпретировать осторожно.

В спецификации с двумя инструментами дополнительно проведён тест Саргана:

- статистика: **1.328**;
- p-value: **0.249**.
Оснований отвергать совместную валидность инструментов не получено.

## Основные результаты

Модель показала, что при прочих равных:

- **возраст автомобиля** отрицательно связан с ценой;
- **пробег** отрицательно связан с ценой;
- **мощность двигателя** положительно связана с ценой;
- автомобили с **правым рулём** в среднем оцениваются дешевле;
- после учёта возможной эндогенности пробега его отрицательная связь с ценой сохраняется и становится более выраженной.
